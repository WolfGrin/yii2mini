сайт для скачивания и установки Yii2 - composer create-project yiisoft/yii2-app-basic ./ 2.0.11

Для установки используя композер (рекомендованно):
!рекомендую закрыть все приложения, которые както могут исопльзовть композер или дебагер. В частности PhpStorm!

1. необходимо иметь свежую версию комопзера: в консоле пишим команду -> php c:\openserver\modules\php\PHP-5.6\composer.phar self-update

2. необходимо быть зарегистрированным на гитхабе, так-же возможно прийдется сгеренировать ключь на гитхабе)

3. открываем консоль и переходим в папку куда надо установить Yii2

4. с сайта копируем, редактируем и пишем в консоль:
(эта строка требуется только при использовании впервые)
------------php composer.phar global require "fxp/composer-asset-plugin:^1.2.0"
->
composer global require "fxp/composer-asset-plugin:^1.2.0"

------------php composer.phar create-project yiisoft/yii2-app-basic basic 2.0.11
->
composer create-project yiisoft/yii2-app-basic ./ 2.0.11

(если консоль запросит ключ, то сгенерированый ключ гитхаба, необходимо вставить в консоль и нажать ентер. При вставке ключя в консоль, он может не отображать!!! Просто жмем ентер!)

Ждем полной установки...

Проверка: переходим в браузере в домен с установленным Yii2 папка /web
"Congratulations!"

Альтернативный спосою (настройка конфигураций вручную):
1. с той же странички качаем "Yii 2 with basic application template"
2. копируем из папки basic содержимое все в домен сайта
3. открываем в папку config ->wweb.php
